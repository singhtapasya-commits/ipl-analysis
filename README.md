# 📊 IPL Performance Analysis: Kohli vs Rohit

## 🧠 Overview
Cricket debates often revolve around comparing legends — but what does the data actually reveal?

This project performs a comprehensive data analysis of the IPL performance of Virat Kohli and Rohit Sharma to identify trends, strengths, and overall impact using statistical and visual techniques.

The goal is to move beyond opinions and provide a data-driven comparison of two of the most prominent players in IPL history.

---

## 🎯 Objectives
- Compare overall batting performance
- Analyze consistency across seasons
- Evaluate strike rate and scoring patterns
- Identify match impact and contribution
- Draw meaningful insights using data visualization

---

## 📂 Dataset
The dataset includes IPL match and player performance data such as:
- Runs scored
- Balls faced
- Strike rate
- Boundaries (4s & 6s)
- Match-wise performance

---

## 🛠️ Tech Stack
- **Python**
- **Pandas** – Data manipulation & analysis  
- **Matplotlib** – Data visualization  
- **Seaborn** – Advanced visualizations  
- **Jupyter Notebook**

---

## 📊 Key Analysis Performed

### 1. 🏏 Total Runs Comparison
- Compared total runs scored by both players across IPL seasons

### 2. ⚡ Strike Rate Analysis
- Evaluated scoring speed and aggression

### 3. 📈 Batting Average
- Measured consistency and reliability

### 4. 🔥 Boundary Analysis
- Compared number of 4s and 6s to assess attacking gameplay

### 5. 📅 Season-wise Trends
- Performance variation across different IPL seasons

### 6. 🎯 Consistency Analysis
- Identified stability using match-wise performance data
